# L1 - Enhanced Todo List

This project is a simple browser-based Todo List with the following features:

## Features
- Add new tasks
- Mark tasks as completed (visually crossed out)
- Delete tasks
- Search tasks in real-time
- Stores tasks in localStorage so they are not lost after refreshing

## How to Use
1. Type a task in the input box and press **Add Task**
2. Use the ✔ button to mark a task as completed
3. Use the ❌ button to delete a task
4. Use the search bar to filter tasks instantly
5. Tasks are saved automatically and restored on page reload

## Technologies Used
- HTML
- CSS
- JavaScript (DOM + localStorage)
